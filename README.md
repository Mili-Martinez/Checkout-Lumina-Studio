# 💡 Lumina Studio — Luxury Checkout Experience

**Lumina Studio** es una interfaz de checkout de alta gama diseñada para una marca ficticia de iluminación sustentable producida mediante impresión 3D. El objetivo de este proyecto fue crear un flujo de compra que refleje la precisión tecnológica y la estética minimalista y sofisticada enfocada en el diseño de autor de los productos.

## 🎯 El Desafío
A diferencia de un e-commerce masivo, Lumina vende objetos de diseño paramétrico. El usuario necesitaba una interfaz que se sintiera tan moderna como el producto, con un enfoque en la claridad, la jerarquía visual y la confianza en el proceso de pago.

## 🛠️ Stack Tecnológico
* **HTML5** (Estructura semántica)
* **Tailwind CSS 4.0** (Estilado rápido y responsive)
* **Antigravity Config** (Para una gestión de diseño fluida)
* **Google Fonts** (Uso de tipografías que comunican modernidad **Boldonse** para una identidad visual fuerte e **Inter** para la legibilidad técnica.)
* **GitHub Pages** (Deployment y hosting)

## 🎨 Estrategia de Diseño y Experiencia
Para **Lumina Studio**, se buscó una identidad que rompa con el esquema tradicional de los formularios de pago, elevando la transacción a una extensión de la marca:

* **Sofisticación Funcional:** Diseño centrado en la "limpieza visual" para elevar la percepción de valor del producto. Se reemplazaron los elementos gráficos tradicionales por una grilla rigurosa y una paleta monocromática técnica, logrando una interfaz que se siente tan artesanal y precisa como las propias lámparas de la colección.
* **Curaduría Tipográfica:** La elección tipográfica actúa como el principal elemento de identidad. El uso de espaciados amplios y jerarquías claras comunica elegancia y vanguardia tecnológica sin necesidad de ornamentos adicionales.
* **Micro-interacciones de Valor:** Se implementaron transiciones suaves (`ease-in-out`) y estados de interacción táctiles (hovers y escalas sutiles) para guiar al usuario, mejorando la usabilidad y la respuesta emocional durante el checkout.
* **Responsive First:** Diseño adaptativo que prioriza la fluidez de la compra tanto en dispositivos móviles como en desktop.
